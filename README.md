# AI Tools Launchpad

A Cloudflare Worker tool site inspired by AmberTreelet's AI tool-site tutorial. It helps makers validate an AI tool-site idea from one keyword: demand signals, target users, MVP scope, SEO pages, monetization, and launch checklist.

## Local dev

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Production domain: <https://tools.efwmcstyle.ccwu.cc>
